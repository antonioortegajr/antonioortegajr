## :computer: Run Locally
                                 
### From root directory :open_file_folder:
                                 
`npm i`
                                 
Run on local
                                 
`npm run dev`
                                 
Visit: http://localhost:3000/
                                 
### Via Docker :whale:
                                 
From directory root
                                 
`docker compose up`
                                 
Visit: http://localhost:3000/
                                 
## Tests
                                 
Jest for testing
                                 
`npm run test`
